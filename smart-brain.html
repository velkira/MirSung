React

Learning React has been a wonderful experience. Previously I learned how to integrate APIs such as a weather app. From school, I was introduced to machine learning in identification. So, I wanted to use a recognition API on my own app. The process also taught me how convenient using React is due to its great documentation and vast online resources.

The app itself takes an image link such as .jpeg or .png and identifies a face using the Clarifai face recognition API. Once a face is detected, the app boxes around the face.
